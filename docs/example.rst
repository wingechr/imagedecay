Example
=======

.. image:: _static/example/samj.jpg
   :width: 80 %
   :align: center
   
and this config file:

.. literalinclude:: _static/example/samj.json

This will apply 

* a little bit of random noise
* some blurring
* some color reduction
* and some random offset overlay

for each iteration.
